## TIC TAC TOE

Ths is a simple terminal based 2 - player tic tac toe game made with python.

# Functions

drawBoard() : Prints the board to the terminal in an aesthetic way.
moves(symbol) : takes position of the symbol as input and updates the table accordingly.
checkWin() : checks if a player won after every move.
checkDraw() : checks if the game ended in a draw

# How to run

1. Clone the repo

git clone https://github.com/shezanusdf/tic-tac-toe

2. Run the game

python tic_tac_toe.py

# How to play

- The program asks you for input as 1-9.
- 1-9 are the positions on the board.
- 1 being the top left corner and 9 being the down right corner.
- play until someone wins or the match ends in a draw.
